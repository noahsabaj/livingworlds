# lw_platform

Platform-specific integrations for Living Worlds.

## Purpose

Handles all platform-specific code and external service integrations, 
keeping the core game logic platform-agnostic.

## Current Features

- Steam API integration (planned)
  - Achievements
  - Cloud saves
  - Multiplayer lobby
  - Workshop support

## Future Platforms

Could expand to support:
- GOG Galaxy
- Epic Games Store
- Console platforms (via Bevy's cross-platform support)
